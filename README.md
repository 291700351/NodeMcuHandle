# 使用NodeMcu制作智能小车控制手柄

## 硬件选择

- NodeMcu-ESP8266   :   WiFi控制
- 按键4个
- 电阻4个

![硬件连接图](./images/image.jpg "硬件连接图")

## 思路原理

4个按键控制左上右下方向
